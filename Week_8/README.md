## 安裝步驟

```
//- 安裝 node.js 跳過

1. $ npm install --global gulp //- 安裝全域 gulp, 如果無法安裝請在最前方加上 sudo -> 例：sudo npm i gulp -g

2. $ git clone 

3. $ cd Hexschool-web-layout-training 

4. $ cd Week_8 // 本次作業在第八週

5. $ npm install // 安裝套件

6. $ gulp // 輸入 gulp 後面如果沒帶名稱會自動抓 gulp 的 default 任務(Gulp 介紹請點參考下方連結處)

7. Enjoy!

```
